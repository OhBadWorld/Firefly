# 前端 + AI 方向的技术积累和项目经验，需要哪些核心能力，可以独立完成哪些 AI 相关的前端落地工作?

考察点

- 能力分析，包括AI提效、AI Agent 开发
- 落地经验
- 框架掌握

1. 掌握大模型的基本原理、提示词工程最佳实践，能针对不同业务场景设计高准确率的提示词模板
2. 能基于 Langchain.js 独立搭建 RAG 私有知识库问答系统，完成从文档处理【文档切片】、向量检索【向量数据库选型】到前端交互的全流程落地
3. 深入理解 Agent 的核心工作流程，能熟练使用 Function Call 实现大模型与业务接口的联动，完成复杂业务场景的 AI 应用开发4. 能独立完成前端 + AI 全栈项目的开发，包括对话交互界面、流式输出渲染、前后端联调、性能优化、异常处理全流程
4. 多智能体协作，上下文，了解多 Agent 协作的核心逻辑，能使用LangGraph、AutoGen 等主流框架实现多智能体的业务落地
5. 能独立完成前端 + AI 业务需求的全流程拆解，包括需求分析、AI 方案设计、技术选型、开发落地、测试优化全链路工作
