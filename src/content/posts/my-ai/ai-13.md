# 基于 Langchain.js 实现一个前端 RAG 应用，能够从本地知识库中检索相关信息并生成回答。要求支持 PDF、Markdown 等多种文档格式，实现高效的文档处理和检索功能。

考察点

- RAG 流程，文档加载 → 切分 → 向量化 → 存储 → 检索 → 生成
- 文档处理、切分策略
- 向量数据库选择
- 检索算法优化

## 思路

RAG 应用的实现需要遵循完整的技术流程:

- 文档的处理阶段

**文档加载阶段**:需要支持多种文档格式的解析，包括 PDF、Markdown、文本文件等。可以使用Langchain.js 提供的各种Loader，如 PyPDFLoader 用于PDF 解析、MarkdownLoader 用于 arkdown 文件解析等。对于特殊格式的文档，可能需要自定义 Loader 来实现解析逻
辑。

**文档切分阶段**:这是影响检索效果的关键步骤。需要将长文档切分成合适大小的块(chunks)，通常建议块大小为100-2000 个字符，并设置 10%-20% 的重叠量(chunkoverlap)以保持语义的连续性。切分策略可以采用递归切分方法，优先在段落分隔符、标题等位置进行切分，避免在句子中间断开。

**向量化**:是将文本转换为向量表示的过程，这使得计算机能够理解文本的语义。可以使用 0penAIEmbeddings、HuggingFaceEmbeddings;等模型来生成文本的向量表示。向量的维度通常为 1536维(0penAI 的 text-embedding-ada-002 模型)，这个维度能够很好地捕捉文本的语义信息。

**向量存储**:需要选择合适的向量数据库。在前端场景下，可以使用 chroma、Pinecone 等轻量级的向量数据库。Chroma 适合本地部署和小规模应用，而Pinecone 则适合大规模和分布式场景。向量数据库需要支持高效的相似性搜索，通常使用余弦相似度或点积来计算向量之间的相似度。

向量数据库，pg-vecotr、chroma、pinecone、qdrat

- 文档的检索阶段

**检索阶段**:需要实现高效的相似性搜索算法。可以使用近似最近邻搜索(ANN)算法来提高搜索速度，如 HNSW(分层可导航小世界)算法。同时要实现多路召回策略，结合关键词搜索和语义搜索，提高检索的准确性。

向量检索、全文检索、混合检索

**生成阶段**:需要将检索到的相关文档作为上下文，与用户问题一起发送给 LLM 生成最终回答。这个过程中要注意上下文窗口的限制，确保总 Token 数不超过模型的限制。可以使用文档摘要或关键信息提取的方法来减少上下文的大小。

## 考查知识点

- RAG 技术的完整流程和原理
- 文档处理和切分技术
- 向量表示和相似性计算
- 向量数据库的选择和使用
- 检索算法优化和结果排序

#### 1) 文档加载与切分

```typescript
// lib/rag/document-processor.ts
import { PDFLoader } from "@langchain/community/document_loaders/fs/pdf";
import { TextLoader } from "langchain/document_loaders/fs/text";
import { DirectoryLoader } from "langchain/document_loaders/fs/directory";
import { RecursiveCharacterTextSplitter } from "@langchain/textsplitters";
import { Document } from "@langchain/core/documents";

export class DocumentProcessor {
  private textSplitter: RecursiveCharacterTextSplitter;

  constructor() {
    this.textSplitter = new RecursiveCharacterTextSplitter({
      chunkSize: 1000,
      chunkOverlap: 200,
      separators: ["\n\n", "\n", "。", "，", "！", "？", " ", ""],
      lengthFunction: (text) => text.length,
    });
  }

  // 加载目录下所有文件
  async loadFromDirectory(directoryPath: string): Promise<Document[]> {
    const loader = new DirectoryLoader(directoryPath, {
      ".pdf": (path = new PDFLoader(path)),
      ".md": (path = new TextLoader(path)),
      ".txt": (path = new TextLoader(path)),
    });
    return loader.load();
  }

  // 处理单个文件上传
  async processFile(file: File): Promise<Document[]> {
    let loader: any;
    if (file.type === "application/pdf") {
      const arrayBuffer = await file.arrayBuffer();
      loader = new PDFLoader(new Blob([arrayBuffer]));
    } else if (file.type === "text/markdown" || file.name.endswith(".md")) {
      const text = await file.text();
      return this.splitDocuments([
        new Document({ pagecontent: text, metadata: { source: file.name } }),
      ]);
    } else {
      const text = await file.text();
      return this.splitDocuments([
        new Document({ pagecontent: text, metadata: { source: file.name } }),
      ]);
    }

    const docs = await loader.load();
    return this.splitDocuments(docs);
  }

  // 文档切分
  async splitDocuments(documents: Document[]): Promise<Document[]> {
    return this.textSplitter.splitDocuments(documents);
  }
}

export const documentProcessor = new DocumentProcessor();
```

### 2)向量存储与检索

```typescript
// lib/rag/vector-store.ts
import { MemoryVectorStore } from 'langchain/vectorstores/memory'
import { OpenAIEmbeddings } from '@langchain/openai';
import { Document }from '@langchain/core/documents';

export class RAGVectorStore {
    private vectorStore: MemoryVectorStore | null = null;
    private embeddings: OpenAIEmbeddings;

    constructor(){
        this.embeddings =new OpenAIEmbeddings({
            modelName: 'text-embedding-3-small',
            dimensions: 1536,
        });
    }

    // 初始化向量存储
    async fromDocuments(uments:Document[]):Promise<void> {
        this.vectorStore= await MemoryVectorStore.fromDocuments(
            documents,
            this.embeddings
        );
    }

    // 添加文档
    async addDocuments(documents: Document[]):Promise<void> {
        if(!this.vectorStore){
            await this.fromDocuments(documents);
            return;
        }
        await this.vectorStore.addDocuments(documents);
    }

    // 相似度搜索
    async similaritySearch(query:string,k:number = 4):Promise<Document[]> {
        if(!this.vectorStore){
            throw new Error('Vector store not initialized');
        }
        return this.vectorStore.similaritySearch(query,k);
    }

    // 带分数的搜索
    async similaritySearchWithScore(query: string, k: number = 4): Promise<[Document, number][]> {
        if(!this.vectorStore){
            throw new Error('Vector store not initialized');
        }
        return this.vectorStore.similaritySearchWithScore(query,k)
    }

    // 检索增强生成:搜索 +上下文构建
    async retrieveContext(query:string,k:number =4):Promise<{
        context: string;
        sources: string[];
    }> {
        const docs = await this.similaritySearch(query, k);
        const context =docs.map((doc，i) => `[文档 ${i + 1}]:${doc.pagecontent}`).join('\n\n');
        const sources =[...new Set(docs.map(doc = doc.metadata.source))];
        return { context,sources };
    }
}

export const ragVectorStore = new RAGVectorStore()
```

#### 3) RAG 问答链

```typescript
// lib/rag/qa-chain.ts
import { ChatPromptTemplate }from'@langchain/core/prompts';
import { StringOutputParser } from '@langchain/core/output_parsers';
import { RunnableSequence, RunnablePassthrough } from '@langchain/core/runnables'.
import { llm }from '../langchain/setup';
import { ragVectorStore }from'./vector-store';

// RAG 问答提示词模板
const RAG_QA_PROMPT = ChatPromptTemplate.fromTemplate(`
  基于以下上下文信息回答用户的问题。如果上下文信息不足以回答问题，请诚实说明"根据现有资料无法回答此问题"。

  上下文:
  {context}

  用户问题: {question}

  回答要求:
  1. 只用中文回答1.
  2. 回答要准确、简洁、专业2.
  3. 只基于提供的上下文，不要编造信息高
  4. 如果上下文中有相关信息，请在回答结尾标注参考来源

  答案:
`)

export class RAGQASystem {
    private qaChain: any;

    constructor() {
        this.initChain();
    }

    private initChain(){
        const retriever = ragVectorStore.SimilaritySearch.bind(ragVectorStore);
        this.qaChain = RunnableSequence.from([
            RunnablePassthrough.assign({
                context: async(input: { question: string }) => {
                    const { context }= await ragVectorStore.retrieveContext(input.question);
                    return context;
                },
            }),
            RAG_QA_PROMPT,
            llm,
            new String0utputParser()
        ]):
    }

    async ask(question:string):Promise<{answer: string;
    sources: string[];
    }> {
        const [answer, { sources }]= await Promise.all([
            this.qaChain.invoke({ question }),
            ragVectorStore.retrieveContext(question)
        ]);
        return { answer,sources };
    }

    async askStream(question: string): Promise<ReadableStream<string>> {
        const encoder =new TextEncoder();
        const { context, sources }= await ragVectorStore.retrieveContext(question);

        const stream = await llm.stream(await RAG_QA_PROMPT.format({ question, context }));

        return new ReadableStream({
            async start(controller){
                for await(const chunk of stream){
                    controller.enqueue(encoder.encode(chunk.content as string));
                }
                // 可选:最后添加来源信息
                // controller.enqueue(encoder.encode(`\n|n参考来源:${sources.join(',')}`));
                controller.close();
            },
        });
    }
}

export const ragQASystem = new RAGQASystem();
```

#### 4) 前端文档上传组件

```tsX
// components/DocumentUploader.tsx
'use client';

import {useState }from 'react';
import { documentProcessor }from '@/lib/rag/document-processor';
import {ragVectorStore }from '@/lib/rag/vector-store'.

export default function DocumentUploader() {
const [uploading, setUploading] = useState(false);
const [uploadedFiles, setUploadedFiles] = useState<string[]>([]);

const handleFileUpload = async (e: React.ChangeEvent<HTMLInputElement>) => {
    const files = e.target.files;
    if (!files?.length)return;

    setUploading(true);

    try {
        for (const file of Array.from(files)) {
            const docs = await documentProcessor.processFile(file);
            await ragVectorStore.addDocuments(docs);
            setUploadedFiles(prev => [...prev,file.name]);
        }
    } catch (error){
        console.error('Upload error:',error);
    } finally {
        setUploading(false);
    }
};

return (
    <div className="p-4 border-2 border-dashed rounded-lg">
      <input
        type="file"
        multiple
        accept=".pdf,.md,.txt"
        onChange={fhandleFileUpload}
        disabled={uploading}
        className="w-full"
      />
        { uploading && <p className="mt-2 text-sm text-gray-500">上传中 ...</p>}
        { uploadedFiles.length > 0 &&(
        <div className="mt-4">
            <h4 className="font-medium mb-2">已上传文件:</h4>
            <ul className="text-sm text-gray-600 space-y-1">
                { uploadedFiles.map(name =>(
                    <li key={name}>✔ {name}</li>
                )) }
            </ul>
        </div>
        )}
    </div>
  );
}
```
