# 设计一个基于 Langchain.js 的 AI 问答系统架构，要求支持多轮对话、记忆管理和工具调用。请详细说明核心组件的设计思路和技术选型。

考察点

- Langchainjs 核心概念，Model、Prompt、indexes、Memory
- 最佳实践

## 思路

- 业务层(Next/Node/Nest/java/python)
- 逻辑编排层(langgraph)
- 组件层(langchain Chain/Memory/RAG)
- 核心层(runnable)
- 模型层(ChatOpenAI/Chat0llama)

对于核心组件的设计，Models 模块需要处理与各种大语言模型的交互，包括0penAI、Claude 等不同供应商的统一接口。Prompts模块要实现提示词的模板化管理，支持动态变量注入。Memory 模块是实现多轮对话的关键，需要选择合适的记忆策略，如 BufferMemory 用于简单的对话历史ConversationSummaryMemory 用于长对话的摘要记忆。

通用型智能体，DeepAgents，做 Harness Engineering

#### 1) 基础配置与模型初始化（AI层的开发）

```typescript
// iib/langchain/setup.ts
import { ChatOpenAI } from '@langchain/openai';
import { chatPromptTemplate,MessagesPlaceholder } from '@langchain/core/prompts';
import { BufferMemory }from 'langchain/memory';
import { RunnableSequence, RunnablePassthrough } from '@langchain/core/runnables';
import { String0utputParser } from '@langchain/core/output_parsers';

// 1、初始化大模型
export const llm = new ChatOpenAI({
    modelName: 'gpt-4o',
    temperature: 0.7,
    streaming: true,
});

// 2.创建对话记忆(多轮对话)
export const memory = new BufferMemory({
    memoryKey:'chat_history',
    returnMessages: true,
})

// 3、构建提示词模板
export const chatPrompt =ChatPromptTemplate.fromMessages([
    [
        'system',
        '你是一个专业的前端 AI 助手，擅长回答 React、Next.js、TypeScript 相关问题。' +
        "请用简洁、专业的中文回答。如果问题超出你的知识范围，请诚实说明。"
    ],
    new MessagesPlaceholder('chat history'),
    ['human','{input}'],
]);


// 4.使用 LCEL 构建对话链
export const conversationChain= RunnableSequence.from([
    RunnablePassthrough.assign({
        chat_history:async(input:{input:string}){
            const { history }= await memory.loadMemoryVariables({});
            return history;
        },
    }),
    chatPrompt
    llm,
    new String0utputParser(),
]);

// 5.包装对话函数，自动保存记忆
export async function chat(input: string){
    const result = await conversationChain.invoke({ input });
    await memory.saveContext({ input },{ output: result });
    return result;
};

```

#### 3) Next.js Server Action 集成（界面层，应用层的开发）

```typescript
// app/actions/chat.ts
'use server';

import { RunnableLambda }from '@langchain/core/runnables';
import { conversationchain,memory } from '@/lib/langchain/setup';

export async function streamChat(message:string){
    const encoder =new TextEncoder();

    return new ReadableStream({
        async start(controller){
            try {
                // 加载历史消息
                const { chat_history }= await memory.loadMemoryVariables({});

                // 流式执行
                const stream =await conversationChain.stream({
                    input: message,
                    chat _history,
                });

                Let fullResponsex = '';
                for await(const chunk of stream) {
                    fullResponse += chunk;
                    controller.enqueue(encoder.encode(chunk));
                }

                // 保存对话记忆
                await memory.saveContext({ input: message },{ output: fullResponse });
                controller.close();
            } catch(error){
                console.error('Stream error:'error);
                controller.error(error);
            }
        },
    });
}

// 清除历史记录
export async function clearChatHistory(){
    await memory.clear();
}
```
