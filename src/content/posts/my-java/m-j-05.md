# 一句话总结面向对象

把一组数据结构和处理它们的方法组成对象（object），

　　把相同行为的对象归纳为类（class），

　　通过类的封装（encapsulation）隐藏内部细节，

　　通过继承（inheritance）实现类的特化（specialization）／泛化（generalization），

　　通过多态（polymorphism）实现基于对象类型的动态分派（dynamic dispatch）。
